## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash

Remember update MONGO URL and Redis

# seed data
$ pnpm run seed

# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Run tests

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
